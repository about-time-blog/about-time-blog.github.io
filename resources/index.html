Resources on Time and Dates

There are some great resources available online when it comes to time and date related things. I have tried to collect the ones that I always seem to return to.

General Resources

TimeAndDate.com The first place to look for answers to any time related question.
The website contains useful information about time zones, daylight saving time, world clock and time zone converters, and much, much more.
World Time Server Any time, anywhere.
Similar to TimeAndDate.com, but they also have a paid time zone database.
ISO 8601 The international standard covering the exchange of date- and time-related data. Use it whenever communicating in text between systems internally and externally.

Time Zones

TimeAndDate.com TimeAndDate have a very useful time zone world map, a zime zone abbreviation list, and a time zone converter for comparing times across the world.
Time.is Shows the current time at a given location, along with the IANA time zone used there.
IANA Time Zones Contains a comprehensive list of IANA time zones.
Windows Time Zones A comprehensive list of Windows time zones in different countries.

.NET

Noda  Time Noda Time is a better date and time API for .NET - or as I like to say: the only sane way to handle date and time in .NET.
Date and Time Data Types in SQL Server An overview of the different date and time data types supported in SQL Server. Make sure to persist your data in the right scope!

Blogs

Jon Skeet’s Coding Blog The famous Jon Skeet is also the main author of Noda Time. You will often see him blogging about it here.
Code of Matt A time-related blog by Matt Johnson, the king of time and one of the other guys behind Noda Time.

Courses and Tutorials

Date and Time Fundamentals A PluralSight course on the fundamentals of date and time by Matt Johnson. If you think you know everything about time, then this course will likely show you wrong.

Miscellaneous

Random.org Random.org has some different online random value generators. They also have a random clock cime generator and a random calendar date generator.
